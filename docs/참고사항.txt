다음 프롬프트를 사용하여 AI에게 프로그램 생성을 요청하였다.

- ta wrapper class 생성

- 옵션으로 기간(start, end 등), 코인 지정(config 파일, 또는 실행시) 등 다양한 옵션 제공

# 3. 전략 생성 규칙

지표는 ta의 지표 활용

- 전략은 커스텀으로 직접 구현
- AI가 자동생성 가능(추후 구현)

첫번째 커스텀 전략 - BB 활용

사용 지표

- Bollinger Bands: BB, BB%, BB width
- Volume
- SMA
- EMA
- MACD
- RSI

상세 전략

- 횡보장 반전매매
- 추세장 추세추종

반전매매

- 횡보장: BB width가 대체로 일정하게 유지
- 추세장: BB width가 지속적으로/급격하게 작아지거나 커짐

RSI < 30 AND price < Bollinger lower → BUY
MA20 > MA60 → BUY
MACD crossover → BUY

# 4. 백테스트 규칙

백테스트 기간

- 기본 2년
- 지정 가능

백테스트 옵션

- 코인 지정 가능: 기본 - config 설정

매매 설정

- Binance, ByBit 선물 거래 / Upbit 현물 거래
- 최저 거래: 거래소 최저 거래로(예시 BTC - 0.002BTC)
- 레버리지: 코인별 차등(예시 BTC - 50배, ETH - 35배) - 상황에 따라 변경
- 거래모드 cross
- 기본 거래 옵션: 물타기 5회까지 가능, 청산: 2~3회 분할 익절, 손절
- stoploss: 레버러지 적용 후 cross모드(전체 마진에서 20%손실시)
- takeprofit: 레버리지 적용 후 개별 포지션 수익 목표 달성 시 50%익절, 나머지 50%는 익절 후 최고가 대비 레버리지 적용 후 20% 하락 시 전부 청산/50% 익절가 이하로 떨어질 경우 전부 청산

전략 필터 조건
Sharpe Ratio > 1.5
Max Drawdown < 20%
Win Rate > 50%
Profit Factor > 1.3
