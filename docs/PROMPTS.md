# 3. 전략 생성 규칙

지표는 ta의 지표 활용

- 전략은 커스텀으로 직접 구현
- AI가 자동생성 가능(추후 구현)

첫번째 커스텀 전략 - BB 활용

사용 지표

- Bollinger Bands: BB, BB%, BB width
- Volume
- SMA
- EMA
- MACD
- RSI

상세 전략

- 횡보장 반전매매
- 추세장 추세추종

반전매매

- 횡보장: BB width가 대체로 일정하게 유지

TIMEFRMAE은 기본 1h 기준으로 long, short 포지션 진입, 청산 시그널 생성
BB width 크기가 레버리지 적용 기준 전체 마진금액을 넘어서면 레버리지 50 -> 40 -> 30 -> 25로 수정
즉, 레버리지 min=25, max=50(BTC 기준)

BB%가 70%일때 1차 매도, 80%: 2차 매도, 90% 3차 매도, 100% 4차 매도, 110% 5차 매도, 120% 손절 50%, 130% 손절 100%
BB%가 30%일때 1차 매수, 20% 2차 매수, 10% 3차 매수, 0% 4차 매수, -10% 5차 매수, -20% 손절 50%, -30% 손절 100%

- 추세장: BB width가 지속적으로/급격하게 작아지거나 커짐

macd, volume, rsi 등 다른 지표도 추가로 체크해서 추세지속/반전을 판단해서 추세지속이면
